OllyPGM
=======

A simple, lightweight OCN testing solution. Now in OllyCode.

Compilation
-----------
We use maven to handle our dependencies.

* Install [Maven 3](http://maven.apache.org/download.html)
* Check out this repo and: `mvn clean install package`
